Esta script permite hacer galerias a FULL SCREEN

Para poner el titulo, subtitulo y pies de fotos hay que llenar el archivo data.XML 
Dependiendo de cuantas fotos existan en la carpeta fotos se deberan poner las siguientes etiquetas

<tituloFoto>TITULO FOTO.</tituloFoto>


El orden en que se llene el XML sera el orden de presentación así el archivo con nombre foto1.jpg correspondera con la primera etiqueta de titulo y así sucesivamente.

El tamaño de las fotografías es de 1600 x 1200, los thumbnail son generados automaticamente.
